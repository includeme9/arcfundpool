# ArcFundPool

ArcFundPool is a mobile-first Web3 MVP for transparent USDC funding pools on Arc Testnet.

## Quick Start

```bash
pnpm install
pnpm dev
```

Open `http://localhost:3000`.

## Notes

- Financial truth is designed to come from onchain state.
- Supabase tables are represented as cache/metadata types and query placeholders.
- Replace TODO config values for Arc RPC, the ArcFundPool contract address, and USDC token address before deployment.
